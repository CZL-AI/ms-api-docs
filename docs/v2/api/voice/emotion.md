# AI声音情绪识别服务

## 产品介绍
---
AI声音情绪识别服务是一款专为宠物主人、宠物行为研究者、宠物服务机构和动物保护组织设计。该服务能够通过分析上传的宠物声音片段，自动识别宠物当前的情绪状态，并提供详细的情绪分析、行为解读和互动建议。系统对上传的声音文件进行智能分析，能够识别多种宠物（主要为犬类和猫类）的情绪状态，如兴奋、焦虑、警觉、寻求关注、孤独等，并提供针对性的互动建议。无论是日常陪伴还是行为训练，系统都能帮助用户更好地理解和回应宠物的需求，提升人宠互动质量。

该服务支持多种音频格式（如wav、mp3等），能够准确识别单次声音片段中的情绪状态，并提供科学的互动指导建议，帮助宠物主人建立更加和谐的人宠关系。

## 使用场景
---
- **宠物日常陪伴**：宠物主人可以通过录制宠物的叫声，了解宠物的情绪需求，提供及时的关爱和互动
- **宠物行为训练**：宠物训练师可利用该服务分析宠物在不同情境下的情绪反应，制定个性化训练方案
- **宠物医疗服务**：兽医和宠物医疗机构可通过情绪分析辅助诊断宠物的健康状况和心理状态
- **宠物寄养服务**：宠物寄养机构可使用该服务监测寄养宠物的情绪变化，提供更贴心的照护服务
- **宠物研究教育**：动物行为研究机构和教育单位可将此服务用于宠物行为学研究和教学实践

## 调用接口：
**请求方式：** `POST（HTTPS）`  
**请求地址：** `https://ms-ai.chongzhiling.com/api/v2.0/ai-b/ai-voice/analysis?token=[ACCESS_TOKEN]`

## **参数列表：**[multipart/form-data]
| 参数  | 类型 | 含义                   |
| ----- | ---- | ---------------------- |
| file | `file` | 上传的音频文件【wav, mp3等格式】

## 返回结果

```json
{
    "data": "情绪状态：警觉或寻求关注\n互动建议：轻轻呼唤它的名字，用平静的语气安抚它，并观察它是否需要外出、进食或只是想得到你的关注。如果它持续吠叫，可以尝试转移它的注意力，例如提供一个它喜欢的玩具。",
    "message": "Get successfully.",
    "success": true
}
```

## 返回参数说明

| 字段名             | 类型     | 说明                                                                 |
|--------------------|----------|----------------------------------------------------------------------|
| `success`             | boolean      | 响应状态码，true 表示请求成功                                       |
| `message`          | string   | 响应描述                                                            |
| `data`             | string   | 包含情绪识别结果和互动建议的文本信息                                |
