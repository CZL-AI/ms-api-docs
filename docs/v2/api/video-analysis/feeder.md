# AI宠物进食进水视频识别服务

## 产品介绍
---
AI宠物进食进水视频识别服务是一款专为宠物主人、宠物行为研究者、宠物服务机构和动物保护组织设计的专业视频分析工具。该服务能够通过分析上传的宠物视频片段，自动识别宠物的进食和饮水行为，并提供详细的行为分析数据。系统对上传的视频文件进行智能分析，能够准确识别宠物在视频中的进食、饮水、未进食、未饮水等行为状态，并提供置信度评分和行为统计信息。该服务支持多种视频格式（如mp4、avi等），能够帮助用户科学监测宠物的饮食习惯，及时发现饮食异常，为宠物健康管理提供数据支持。

该服务可广泛应用于宠物日常监护、健康评估、行为研究等场景，帮助用户全面了解宠物的饮食规律和健康状况。

## 使用场景
---
- **宠物日常监护**：宠物主人可以通过上传宠物进食视频，监测宠物的饮食情况，及时发现食欲不振或异常行为
- **宠物健康管理**：兽医和宠物医疗机构可通过分析进食饮水行为，辅助诊断宠物的健康状况和营养需求
- **宠物寄养服务**：宠物寄养机构可使用该服务记录和监测寄养宠物的饮食情况，向宠物主人提供详细的照护报告
- **宠物行为研究**：动物行为研究机构可利用该服务进行宠物饮食行为模式研究，收集科学数据
- **宠物营养评估**：宠物营养师可使用该服务分析宠物的进食习惯，制定个性化的营养方案

## 调用接口：
**请求方式：** `POST（HTTPS）`  
**请求地址：** `https://ms-ai.chongzhiling.com/api/v2.0/ai-b/ai-feeder/predict?token=[ACCESS_TOKEN]`

## **参数列表：**[multipart/form-data]
| 参数  | 类型 | 含义                   |
| ----- | ---- | ---------------------- |
| file | `file` | 上传的视频文件【mp4, avi等格式】 |

## 返回结果

```json
{
    "data": {
        "conf": 0.59,
        "counts": 4,
        "index": 1,
        "label": "进食",
        "total_segments": 4
    },
    "message": "Get successfully.",
    "success": true
}
```

## 返回参数说明

| 字段名             | 类型     | 说明                                                                 |
|--------------------|----------|----------------------------------------------------------------------|
| `success`          | boolean  | 响应状态码，true 表示请求成功                                       |
| `message`          | string   | 响应描述                                                            |
| `data`             | object   | 包含行为识别结果的数据对象                                          |
| └─`conf`           | float    | 识别置信度（范围：0~1）                                             |
| └─`counts`         | int      | 当前行为类型出现次数                                                |
| └─`index`          | int      | 当前片段在视频中的索引                                              |
| └─`label`          | string   | 识别的行为标签（如"饮水"、"进食"、"其他"、"假动作"）              |
| └─`total_segments` | int      | 视频被分割的总片段数                                                |
