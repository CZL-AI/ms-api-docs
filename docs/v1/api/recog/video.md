## 视频参数
| 项目         | 要求                            | 示例值           | 说明                 |
| ---------- | ----------------------------- | ------------- | ------------------ |
| 编码格式   | H.264（AVC）                    | `h264`        | 模型兼容最广的压缩编码        |
| 封装格式   | MP4                           | `.mp4`        | 通用兼容性好             |
| 分辨率    | 720 × 480,1280 × 720  | `720 × 480`   | 输入统一便于模型预处理        |
| 显示比例   | 16:9                          | `16:9`        | 主流显示比例             |
| 帧率     |  5~30 FPS                     | `10/1`        | 一致帧率便于帧采样和时序建模     |
| 时长     | 建议 10 -+3 秒                     | `12.32 秒`     | 控制输入体积，适合批处理或实时性要求 |

## 音频参数
| 项目   | 要求                  | 示例值     |
| ---- | ------------------- | ------- |
| 编码格式 | AAC / PCM           | aac     |
| 采样率  | 16000 Hz 或 44100 Hz | 44100   |
| 通道数  | 单声道或双声道             | 1 或 2   |
| 时长   | 同视频一致               | 12.32 秒 |
| 位深   | 16-bit 或 24-bit     | 16-bit  |
